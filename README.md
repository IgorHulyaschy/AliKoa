# AliKoa Marketplace

![Marketplace pic](docs/marketplace.png)

The idea is to create a marketplace to help people in selling and buying different goods.

## Use case diagram

![Interaction between user and system](docs/SupaDupaMarketplace.png )

Main points:

### User

- can create an account
- can see ratings and reviews
- can browse wide selection of goods
- can login to the account
- can manage profile(edit profile, add ads, remove ads, edit ads)
- can add to cart(update, view cart)
